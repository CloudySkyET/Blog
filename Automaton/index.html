自动机 学习笔记

自动机，全称有限状态自动机（DFADFADFA）是信息学中一类常用的算法。

它包括 5 个要素：

  1. 状态集合 QQQ
  2. 字符集 Σ\SigmaΣ
  3. 状态转移函数 δ\deltaδ 使得 δ(q,σ)=q′(q,q′∈Q,σ∈Σ)\delta(q, \sigma) = q'(q, q'\in Q, \sigma\in \Sigma)δ(q,σ)=q′(q,q′∈Q,σ∈Σ)
  4. 开始状态 s∈Qs \in Qs∈Q
  5. 接收的状态集合 F⊆QF\subseteq QF⊆Q

可以把这个东西看成一张有向图，图中的所有顶点所组成的集合为 QQQ ，图中的边为 δ\deltaδ ，边权只能为 Σ\SigmaΣ 中的一种。sss 为起点。

值得一提的是，并不是所有的自动机都是一棵树，但必然是一个 DAGDAGDAG 。

ACACAC 自动机

ACACAC 自动机是一类以 TrieTrieTrie 为基础，结合了 KMPKMPKMP 思想，用来在线性时间内解决多串同时匹配问题的算法。它通过维护 failfailfail 指针来保证复杂度，类似与 KMPKMPKMP 的 nxtnxtnxt 数组，不同的是，它维护的是全局失配转移路径，即有可能在 AAA 串匹配完成或失败后直接开始进行对 BBB 串的匹配。

具体实现：

failfailfail 指针的含义：设字符串 rootABrootABrootAB 在 TrieTrieTrie 上路径末尾对应的节点 aaa ，则 failafail_afaila​ 指向的便是路径 rootBrootBrootB 的末尾 bbb，且满足 BBB 最大，但并不一定是这条链的结尾。如：一个 TrieTrieTrie 中顺序插入了 ABCABCABC , BCDBCDBCD 和 CCC 则他们的节点编号为 123 456 7 那么 fail3=5fail_3 = 5fail3​=5 .

同时可以得出一个节点 failfailfail 指向的节点深度一定小于当前节点。

ACACAC 自动机的构建：

先把 TrieTrieTrie 树构建出来，然后通过广搜构建 failfailfail 指针。

  • 将第一层所有节点的 failfailfail 指针指向 rootrootroot ，并将它们入队。

    令某个节点父亲节点 failfailfail 指针的相同转移子节点为失配节点。

  • 如果当前节点存在，将它的 failfailfail 指向失配节点。

  • 如果当前节点不存在，将它指向失配节点。

    通过这样的操作，根据广搜的特性，深度小于它的节点一定比它先搜到，所以无论失配节点存不存在，这个操作都是有意义的。

通过上面的构造，就打通了一条从当前节点到 rootrootroot 的路径。也就是说不同于 KMPKMPKMP ，无需判断是否失配直接跳子节点即可。

经典的操作：

  • 查询每个模式串在文本串中出现的次数。

  • 查询出现最多的模式串和出现次数。

    查询时每到一个节点将它的标记 +1+1+1 ，最后将每个点的 failfailfail 向它连边（注意方向），然后跑一边类似树形 DP 的操作，向上统计答案即可，因为如果某个串成功匹配，一定会对它 failfailfail 边指向的节点造成贡献。

    一定要反向连边的原因是所有的节点最终都会 failfailfail 到 rootrootroot ，反向连边方便进行 dfsdfsdfs ，当然也可以进行拓扑排序正向连边，同时统计答案。

后缀自动机 SAMSAMSAM

后缀自动机是一个用途广泛的字符串处理算法。SAMSAMSAM 最重要的，也是它建立的初衷，是它维护了所有子串的信息。具体来说从初始节点 t0t_0t0​ 开始的任意一条路径都与 sss 的子串构成对应关系。但这并不能唯一确定一个自动机，因此它还具有一些其他的性质。

令 endpos(x)endpos(x)endpos(x) 为子串 xxx 在自动机中的所有出现位置。

那么相对于 endpos(x)endpos(x)endpos(x) 只会出现 3 种情况：

endpos(y)⊆endpos(x),当且仅当 x 为 y 的后缀endpos(y)⊇endpos(x),当且仅当 y 为 x 的后缀endpos(y)∩endpos(x)=∅,当且仅当 x 和 y 不构成后缀关系endpos(y) \subseteq endpos(x), \text{当且仅当}\ x \ \text{为}\ y \ \text{的后缀} \\ endpos(y) \supseteq endpos(x), \text{当且仅当}\ y \ \text{为}\ x \ \text{的后缀} \\ endpos(y) \cap endpos(x) = \varnothing, \text{当且仅当}\ x \ \text{和}\ y \ \text{不构成后缀关系} endpos(y)⊆endpos(x),当且仅当 x 为 y 的后缀endpos(y)⊇endpos(x),当且仅当 y 为 x 的后缀endpos(y)∩endpos(x)=∅,当且仅当 x 和 y 不构成后缀关系

（要注意后缀关系与子集关系是反向的）

也有一些子串他们虽然长度不同，但是 endposendposendpos 集合是完全相同的，称他们为等价类，这些子串最终会被合并为一个节点。也就是说后缀自动机上每个节点对应的字符串长度是一个区间。

同时维护后缀链接 fafafa 指向 endposendposendpos 真包含当前节点 endposendposendpos 集的最长的后缀，即 longest(fax),endpos(fax)⫌endpos(x)longest(fa_x), endpos(fa_x) \supsetneqq endpos(x)longest(fax​),endpos(fax​)⫌endpos(x)。

有了这两个性质辅助，就可以唯一确定一个后缀自动机了。

后缀自动机每个节点维护的基本信息：

  • lenlenlen 表示最长等价类长度
  • failfailfail 表示后缀链接
  • sam[σ]sam[\sigma]sam[σ] 表示状态转移边
  • sizsizsiz 表示每个节点出现的次数

跳 samsamsam 边相当于在后面加字符， 跳 failfailfail 边相当于在前面删字符。

后缀自动机的线性构建：

因为 SAMSAMSAM 结构较为复杂，且不确定性因素较多，所以构建分为两步

  1. 逐个插入字符，构建 samsamsam 转移边，长度 lenlenlen 和 failfailfail 指针。
  2. 在搭好结构的 SAMSAMSAM 上求 sizsizsiz。

设上次插入的点为 ppp, 这次要插入的点为 npnpnp，令 lennp=lenp+1len_{np} = len_p + 1lennp​=lenp​+1 ，接下来要进行的操作是给 npnpnp 找 failfailfail 值。

  • 令 ppp 重复跳 fafafa 指针直到找到有相同转移的点或虚拟节点。

  • 如果为虚拟节点， failnp=0fail_{np} = 0failnp​=0 并退出。

  • 否则令当前节点 ppp 的对应转移为 qqq.

  • 可以发现如果当前节点 lenq=lenp+1len_q = len_p + 1lenq​=lenp​+1 ，直接链 failfailfail 并退出。
    因为根据上面的定义，这条边不会被后面操作拆开。

  • 否则：

    • 新建节点 nqnqnq ，并复制 qqq 除 lenlenlen 以外的信息。令 lennq=lenp+1len_{nq} = len_p + 1lennq​=lenp​+1.

      相当于强行把 qqq 删了点东西。

    • 将 failqfail_qfailq​ 和 fanpfa_{np}fanp​ 同时指向 nqnqnq 。

    • 将所有 failfailfail 链接着 qqq 的点的图边全部替换成 nqnqnq 。

这里面最长的串是 npnpnp, 最短的串是 nqnqnq （如果有的话），从 npnpnp 到 qqq 再到 nqnqnq 依次删除了一些前缀。

构建 sizsizsiz 有两种常见的方法。一种是像 ACACAC 自动机一样反着连 failfailfail 边建出 failfailfail 树，然后跑树形 dpdpdp ，另一种是通过特殊性质：lenfailp<lenplen_{fail_p} < len_plenfailp​​<lenp​ ，对映射数组进行时间复杂度为线性的排序来转移。第二种常数较小，且代码相对短。

回文自动机 PAMPAMPAM

回文自动机是一种可以存储一个串中所有回文子串的高效数据结构。同样是一个节点表示一个子串。建出来的形态是两颗树，一颗表示长度为奇数的回文串，

需要维护的东西：

  • lenlenlen 字符串长度。
  • pam[σ]pam[\sigma]pam[σ] 字符边。不同于其他自动机，跳 pampampam 边的含义是在前面和后面都插入当前字符。
  • failfailfail 失配边。仍然是指向当前子串的最长后缀。

PAMPAMPAM 的线性构造：

首先构建两个点 000 和 111 分别代表偶根和奇根，并将 111 号点的长度初始化为 −1-1−1 。

令 ppp 为上一次操作后的节点， qqq 为当前要放入的节点。每插入一个字符不断地令 ppp 跳 failfailfail 直到可以匹配上回文。显然你的 qqq 是要接在 ppp 上的，但是不能着急。先设 np=failpnp = fail_pnp=failp​ ，令 npnpnp 不断跳 failfailfail 直到匹配上回文，将 failqfail_qfailq​ 指向此时的 npnpnp ，然后再将 pamp[σ]pam_p[\sigma]pamp​[σ] 指向 qqq ，同时 lenq=lenp+2len_q = len_p + 2lenq​=lenp​+2。
