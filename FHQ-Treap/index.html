FHQ_Treap 学习笔记

时间复杂度：O(nlogn)O(nlogn)O(nlogn)

FhqTreapFhqTreapFhqTreap 是一种 TreapTreapTreap 的变形，他摒弃了传统平衡树的旋转操作，取而代之的，是分裂和合并操作。具体而言一共包含 333 种函数：

  • 按大小分裂，可以直接退化为非递归形式且不改变树的结构，类似线段树上二分，找到第 kkk 大的数（ kthkthkth ）。

  • 按权值分裂

  • 启发式合并操作

其中平衡树的操作都可以转变为这几种操作的组合。要维护的信息和普通的 TreapTreapTreap 基本一致，随即键值来保证树的结构的思想也如出一辙。

具体实现：

  1. 插入操作：读入一个数 xxx ，将 xxx 放入平衡树中。可以转变先按 xxx 将树分裂为左右两棵树 aaa 和 bbb，先将 aaa 和 xxx 合并，再将合并后的树与 bbb 合并。
  2. （个人认为最难理解）删除操作：读入一个数 xxx， 删除平衡树中第一个权值为 xxx 的点。可以转变为先按 xxx 分裂为左右两棵子树 a,ba, ba,b，再将 aaa 按 x−1x - 1x−1 分裂为 a,ca, ca,c，最后先将 ccc 的左右子树合并再与 a,ba, ba,b 合并。原因：可能有多个相同的值，而只需要删除一个，在这些权值相同的数中，最好找的就是根节点，所以直接删除根节点即可。
  3. 查询排名操作：按 x−1x - 1x−1 分裂，左子树的大小 +1+ 1+1 就是 xxx 的排名。（因为可能存在重复数字，所以不能直接按 xxx 分裂）
  4. 按照排名找数：直接调用函数即可。
  5. 查前驱操作：先按 xxx 分裂为 a,ba, ba,b，再将 aaa 按 x−1x - 1x−1 分裂，或按 aaa 子树大小分裂，右子树即为所求（注意求出答案后要先把 aaa 重新合并好，再将 a,ba, ba,b 合并起来）。
  6. 查后继操作：与前驱类似，先按 xxx 分裂为 a,ba, ba,b，再将 bbb 按 x+1x + 1x+1 分裂，或按 111 大小分裂，左子树即为所求。
