P3558 [POI2013]BAJ-Bytecomputer

P3558 [POI2013]BAJ-Bytecomputer

题目大意：

给定一个长度为 n 的 -1, 0, 1 序列。每次可以将 ax+=ax−1(x∈[2,n])a_x += a_{x-1} (x\in [2, n])ax​+=ax−1​(x∈[2,n]) 求最小的修改次数使给定序列成为不降序列，无解输出 -1。

题目正解：

线性 DP

题目思路：

fi,jf_{i,j}fi,j​ 表示将第 iii 位改为 j−1j - 1j−1 形成非降序列的最少修改次数。

fi,0f_{i, 0}fi,0​ 只能由 fi−1,0f_{i - 1, 0}fi−1,0​ 转移而来，如果：

  • ai=−1a_i = -1ai​=−1, fi,0=fi−1,0f_{i,0} = f_{i - 1, 0}fi,0​=fi−1,0​
  • ai=0a_i = 0ai​=0, fi,0=fi−1,0+1f_{i,0} = f_{i-1, 0} + 1fi,0​=fi−1,0​+1 (把 0 拉成 -1)
  • ai=1a_i = 1ai​=1, fi,0=fi−1,0+1f_{i,0} = f_{i-1, 0} + 1fi,0​=fi−1,0​+1 (把 1 拉成 -1)

fi,1f_{i, 1}fi,1​ 的转移相对复杂，注意上一个数不一定是 0，也可以是 -1。
也要注意一下，如果 ai−1a_{i-1}ai−1​ 为 0 并不能对 aia_iai​ 造成任何改变。

  • ai=−1a_i = -1ai​=−1, fi,1f_{i,1}fi,1​ 失效（显然 0 无法把 -1 拉成 0）
  • ai=0a_i = 0ai​=0, fi,1=min(fi−1,0,fi−1,1)f_{i,1} = min (f_{i-1, 0}, f_{i-1, 1})fi,1​=min(fi−1,0​,fi−1,1​)
  • ai=1a_i = 1ai​=1, fi,1=fi−1,0+1f_{i,1} = f_{i-1, 0} + 1fi,1​=fi−1,0​+1 （这一点比较难想到, -1 把 1 拉成了 0 然后接了上去）

fi,2f_{i, 2}fi,2​ 转移比较暴力，如果 ai≠1a_i \ne 1ai​=1 就拉，否则由 fi−1f_{i-1}fi−1​ 的三个状态取 min 转移（显然 -1, 0, 1 都可以接到 1 上构成非降序列。）

转移方程较多，但并不难想到，注意不要产生丢落即可。

边界的话除了 f1,a1+1=0f_{1, a_1 + 1} = 0f1,a1​+1​=0 以外都设为 inf，最后统计答案如果都大于等于 inf 就无解，否则取 min 。
