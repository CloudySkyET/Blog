P4552 [Poetize6] IncDec Sequence

求出a的差分序列b,其中 b1=a1b_{1}=a_{1}b1​=a1​，bi=ai−ai−1(2≤i≤n)b_{i}=a_{i}-a_{i-1}(2≤i≤n)bi​=ai​−ai−1​(2≤i≤n)。令 bn+1=0b_{n+1}=0bn+1​=0。(一会有大用)

题目对序列a的操作，相当于每次可以选出 b1,b2,…,bn+1b_{1},b_{2},…,b_{n+1}b1​,b2​,…,bn+1​ 中的任意两个数，一个加1，另一个减1。目标是把 b2,b3,…,bnb_{2},b_{3},…,b_{n}b2​,b3​,…,bn​ 变为全零。最终得到的数列 aaa 就是由 nnn 个 b1b_{1}b1​ 构成的。

从 b1,b2,…,bn+1b_{1},b_{2},…,b_{n+1}b1​,b2​,…,bn+1​ 中任选两个数的方法可分为四类:

  1. 选 bib_{i}bi​ 和 bjb_{j}bj​，其中 2≤i,j≤n2≤i,j≤n2≤i,j≤n。这种操作会改变 b2,b3,…,bnb_{2},b_{3},…,b_{n}b2​,b3​,…,bn​ 中两个数的值。应该在保证 bib_{i}bi​ 和 bjb_{j}bj​ 一正一负的前提下，尽量多地采取这种操作，更快地接近目标。
  2. 选 b1b_{1}b1​ 和 bjb_{j}bj​，其中 2≤j≤n2≤j≤n2≤j≤n。
  3. 选 bib_{i}bi​ 和 bn+1b_{n+1}bn+1​，其中 2≤i≤n2≤i≤n2≤i≤n。
  4. 选 b1b_{1}b1​ 和 bn+1b_{n+1}bn+1​，这种情况没有意义，因为它不会改变 b2,b3,…,bnb_{2},b_{3},…,b_{n}b2​,b3​,…,bn​ 的值，相当于浪费了一次操作，一定不是最优解。

设 b2,b3,…,bnb_{2},b_{3},…,b_{n}b2​,b3​,…,bn​ 中正数总和为 ppp，负数总和的绝对值为 qqq。首先以正负数配对的方式尽量执行第1类操作，可执行 min⁡(p,q)\min(p,q)min(p,q) 次。剩余 ∣p−q∣|p-q|∣p−q∣ 个未配对，每个可以选与 bnb_{n}bn​ 或 bn+1b_{n+1}bn+1​ 配对，即执行第2或3类操作，共需 ∣p−q∣|p-q|∣p−q∣ 次。

综上所述，最少操作次数为 min⁡(p,q)+∣p−q∣=max⁡(p,q)\min(p,q)+|p-q|=\max(p,q)min(p,q)+∣p−q∣=max(p,q) 次。根据 ∣p−q∣|p-q|∣p−q∣ 次第2、3类操作的选择情况，能产生 ∣p−q∣+1|p-q|+1∣p−q∣+1 种不同的 b1b_{1}b1​ 的值，即最终得到的序列a可能有 ∣p−q∣+1|p-q|+1∣p−q∣+1 种。

摘自《算法竞赛进阶指南》

知道这些之后，你就会惊奇的发现，这道题不用差分序列！
只要在读入时维护正数的和以及负数和的绝对值就可以了，避免了不必要的空间浪费。
