CDQ分治 学习笔记

CDQ 分治是 OI 中一个比较常用的分治算法。模板的实现较为简单，重点在于领会思想。

CDQ 分治用来解决点对三维偏序问题

点对的三位偏序可以表示为

有 nnn 个元素，第 iii 个元素有 ai,bi,cia_i,b_i,c_iai​,bi​,ci​ 三个属性，设 f(i)f(i)f(i) 表示满足 aj≤aia_j \leq a_iaj​≤ai​ 且 bj≤bib_j \leq b_ibj​≤bi​ 且 cj≤cic_j \leq c_icj​≤ci​ 且 j≠ij \ne ij=i 的 jjj 的数量。求 fif_{i}fi​ 。

先假设没有相同的元素，具体实现：

  • 将整个数组按照 aia_iai​ 排序，开始递归。
  • 将当前区间二分，左右两边分别按照 bib_ibi​ 排序。

现在考虑左边的所有数字 aia_iai​ 一定小于右边，这是在分治开始之前就已经确定好的。

然后由于你在左右分别按照第二维排序了，所以第二维也分别是有序的。

  • 所以就可以在左右分别维护一个指针 pl,prpl, prpl,pr 。

  • 枚举 pr∈[mid+1,r]pr \in[mid + 1, r]pr∈[mid+1,r] 。

  • 跳 pl∈[l,mid]pl\in[l, mid]pl∈[l,mid] 使 bpl<bprb_{pl} < b_{pr}bpl​<bpr​ 始终成立，同时将已经扫过的 cplc_{pl}cpl​ 插入树状数组。

  • 每次当 plplpl 不能再跳时统计 prprpr 贡献 anspr+=ask(cpr)ans_{pr} += ask (c_{pr})anspr​+=ask(cpr​) ，即第三维也比当前数字小的数。

  • 枚举 i∈[l,pl]i\in[l, pl]i∈[l,pl] ，在树状数组上消除贡献。

由于你的前两维已经确定了，所以只要第三维也比当前数小的话，就一定满足条件。

由于以上整个过程都处于分治过程中，所以一定可以保证不重不漏。

另外，如果要在递归之前排序的话，一定要建临时数组，否则递归回出错；

也可以先递归，然后进行操作，最后将当前整段区间进行排序，以便父层使用，最后分治完再将序列排回原样。

两种写法更推荐后者，排序没有过多的复杂度浪费。
