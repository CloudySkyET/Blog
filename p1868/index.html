P1868 饥饿的奶牛

题目大意：

共有 n 个区间，选取任意个不重叠的区间使覆盖面积最大，求最大面积。

题目正解：

DP + 二分

题目思路：

首先考虑给线段排序。因为每回你都要满足左端点大于要转移点的右端点，所以按照右端点排序会方便些，因为这样就可以二分了。

然后考虑转移方程。设 fif_ifi​ 表示考虑到第 i 片草的最大价值。

可以按照这个草选不选分为两种。

如果不选那么 fi=fi−1f_i = f_{i-1}fi​=fi−1​

如果选的话因为 f 满足单调性，所以直接二分查找最后一个右端点小于当前草左端点的草，设它为 k。

fi=fk+valif_i = f_k + val_ifi​=fk​+vali​

将以上两种情况取 maxmaxmax 即可。

最终转移方程：

fi=max(fi−1,fk+ri−li+1)f_i = max (f_{i-1}, f_k + r_i - l_i + 1) fi​=max(fi−1​,fk​+ri​−li​+1)
