卷积与乘法卷积

卷积通俗理解就是求三个函数关系的式子。常见的有加法卷积和乘法卷积，又称狄利克雷卷积。
加法卷积的公式为：

g(n)=∑i=1nf(i)×h(n−i)g(n) = \sum_{i = 1}^n f(i) \times h(n - i) g(n)=i=1∑n​f(i)×h(n−i)

但它不在这次的研究范围内。

乘法卷积的公式为：

g(n)=∑d∣nf(d)×h(n/d)g(n) = \sum_{d|n} f(d) \times h(n/d) g(n)=d∣n∑​f(d)×h(n/d)

为什么数论函数会和这个东西有关系呢？

因为它的求和条件 d∣nd|nd∣n 会产生许多性质。

先来说明三个和卷积有关的函数。

  1. ε\varepsilonε 函数，又称元函数。定义式： ε(1)=1,ε(n)=0(n≠1)\varepsilon(1) = 1, \varepsilon(n) = 0 (n \ne 1)ε(1)=1,ε(n)=0(n=1)
  2. III 函数，又称恒等函数。定义式： I(n)=1I(n) = 1I(n)=1
  3. IdIdId 函数，又称单位函数。定义式： Id(n)=nId(n) = nId(n)=n

这三个函数是乘法卷积的基本函数，接下来只需要用他们 瞎搞 进行一些推算。
