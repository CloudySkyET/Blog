容斥原理常见问题模型

不定方程非负整数解计数

简单版：给定 kkk 个未知数 x1,x2,⋯ ,xkx_1, x_2, \cdots, x_kx1​,x2​,⋯,xk​ 和一个方程 x1+x2+⋯+xk=rx_1 + x_2 + \cdots + x_k = rx1​+x2​+⋯+xk​=r ，求方程非负整数解的个数。

对于这个问题可以直接转换成向 kkk 个盒子里放 rrr 个球，可空的方案数。然后在每个盒子里先补上一个球，变成 kkk 个盒子里放 r+kr + kr+k 个球，不可空的方案数。

然后插板法，在 r+k−1r + k - 1r+k−1 个球之间的空隙里插入 k−1k - 1k−1 个板。总的答案 (r+k−1k−1)\dbinom{r + k - 1}{k - 1}(k−1r+k−1​) 。

升级版：给定 kkk 个未知数 x1,x2,⋯ ,xkx_1, x_2, \cdots, x_kx1​,x2​,⋯,xk​ ，kkk 个约束条件分别为 xk≤nkx_k \le n_kxk​≤nk​ 和一个方程 x1+x2+⋯+xk=rx_1 + x_2 + \cdots + x_k = rx1​+x2​+⋯+xk​=r ，求方程非负整数解的个数。

这是一个求交集的问题，容斥是用来求并集的，所以可以转化为先求所有不满足条件的情况，再用上面算出来的总情况做差。

对于每个未知数 xix_ixi​ ，保证 xix_ixi​ 不满足要求的办法就是用 xi′=xi+ni+1x_i' = x_i + n_i + 1xi′​=xi​+ni​+1 替换 xix_ixi​ 。所以对于任意一个集合 SSS 表示集合中的元素全部不符合条件，则

q(S)=(r+k−1−∑i∈s(ni+1)k−1)q(S) = \dbinom{r + k - 1 - \sum_{i \in s}(n_i + 1)}{k - 1} q(S)=(k−1r+k−1−∑i∈s​(ni​+1)​)

容斥系数 f(∣S∣)f(|S|)f(∣S∣) 很简单，就是 (−1)∣S∣(-1)^{|S|}(−1)∣S∣ 。

所以答案为

ans=∑S⊆Uf(∣S∣)q(S)=∑s⊆U(−1)∣S∣(r+k−1−∑i∈s(ni+1)k−1)ans = \sum_{S \subseteq U}f(|S|)q(S) = \sum_{s \subseteq U}(-1)^{|S|}{r + k - 1 - \sum_{i \in s}(n_i + 1) \choose k - 1} ans=S⊆U∑​f(∣S∣)q(S)=s⊆U∑​(−1)∣S∣(k−1r+k−1−∑i∈s​(ni​+1)​)
