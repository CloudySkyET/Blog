拓展欧几里得算法 学习笔记

拓展欧几里得算法是用来解决二元一次不定方程的一种算法，即这个式子：

ax+by=cax + by = cax+by=c

其中 c 一定要满足 c∣gcd(a,b)c | gcd(a,b)c∣gcd(a,b) 。否则无解。

它实际上是先解出来 ax+by=gcd(a,b)ax + by = gcd(a,b)ax+by=gcd(a,b) ，然后再将解出来的 x ×c/gcd(a,b)\times c / gcd(a,b)×c/gcd(a,b) 即可。

  • 当 b=0b = 0b=0 时，一组解就是 {x=1y=0\begin{cases}x = 1 \\y = 0\end{cases}{x=1y=0​ 。
  • 只要在回溯时倒着推回去便能找到一组特解。
  • x0x_0x0​ 便是 x+k×lcm(a,b)x + k \times lcm(a,b)x+k×lcm(a,b) 。

exgcd 同时也可以用来求乘法逆元，条件是 xxx 和 ppp 互质。

原理是：

设 xxx 在mod  p\mod pmodp 意义下的逆元是 x−1x^{-1}x−1。

∵x∗x−1≡1(mod  p)\because x * x^{-1} \equiv 1(\mod p) ∵x∗x−1≡1(modp)

∴x∗x−1=k∗p+1\therefore x * x^{-1} = k * p + 1 ∴x∗x−1=k∗p+1

即x∗x−1−k∗p=1\text即 x * x^{-1} - k * p = 1 即x∗x−1−k∗p=1

这样就化成了扩展欧几里得的常见形式了，xxx 和 ppp 已知，x−1x^{-1}x−1 和 kkk 是要求的。（但一般 k 求出来没有什么用）
