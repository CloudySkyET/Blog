拓扑排序 学习笔记

时间复杂度:O(n+m)O(n+m)O(n+m)

拓扑排序是对有向无环图节点排序的一种方法，一般通过广度优先搜索实现。可以方便我们实现在 DAG 上 DP 的转移，满足 DP 的无后效性。

具体实现：

  1. 在建图的同时统计每个点的入度。扫一遍所有节点，将入度为零的节点入队。
  2. 每次从队列中还未被访问的节点取出，从图中删除，即将它能到的节点入度 -1。
  3. 将入度被减为 0 的节点入队。
  4. 最终队列里的节点顺序即为此 DAG 的拓扑排序。
